# Descripción
El objetivo es completar la transición estética del sitio web hacia un estilo minimalista monocromático, alineado con el nuevo logotipo negro y blanco. Se debe eliminar cualquier rastro de los colores anteriores (naranja y azul) y asegurar una experiencia visual premium.

# Objetivos
- [ ] Eliminar rastros de colores corporativos antiguos (naranja/azul) en componentes críticos.
- [ ] Ajustar gradientes o sombras para que sean sutiles y profesionales.
- [ ] Asegurar que el logo del Navbar y Footer en formato imagen se visualice correctamente.
- [ ] Normalizar la paleta en todas las páginas (Hero, Servicios, Contacto).

# Requisitos Técnicos
- Archivo `resources/css/app.css` como base de estilo.
- Componentes Blade: Navbar, Footer, Layout App.
- Tailwind 4 (visto en app.css).

# Criterios de Aceptación
- [ ] No existen colores #ff6b00 (naranja) ni #0047ab (azul) en el CSS o inline styles.
- [ ] El contraste blanco/negro/gris cumple con estándares de legibilidad.
- [ ] El sitio refleja una estética de "alta tecnología" y minimalismo.

# Notas / Contexto
Esta es la primera especificación siguiendo el nuevo flujo de Spec-Driven Development.
