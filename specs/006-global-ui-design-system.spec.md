# Descripción
Definir los cimientos visuales de la nueva identidad de Conycom. El objetivo es pasar de una estética genérica a una de **Minimalismo Técnico Suizo (Swiss Design)** que transmita agilidad, modernidad y la confianza necesaria para liderar la evolución digital de una empresa.

# Objetivos
- [x] **Paleta de Colores**: Refinar el monocromo. Negros profundos, blancos crudos y grises técnicos (`#e3e3e0`).
- [x] **Tipografía**: Jerarquía clara basada en **Inter** para legibilidad y cercanía.
- [ ] **Narrativa de Marca**: Enfocar en **"Agilidad para Innovar. Solidez para Escalar."**. El mensaje debe ser aspiracional y centrado en soluciones de negocio, no en títulos personales.
- [x] **Grilla y Espaciado**: Sistema de 8px con bordes finos.

# Requisitos Estéticos
- **Surface**: Alternar fondos blancos con bloques de diseño estructurado para generar dinamismo.
- **Narrativa**: La referencia a la disciplina debe ser corporativa, centrada en el rigor y la metodología de ingeniería.

# Criterios de Aceptación
- [ ] El Hero invita a la acción con un mensaje de crecimiento.
- [ ] La estructura de la página se siente como una "agencia boutique ágil".
- [ ] Los tecnicismos se presentan como el respaldo de la solución.

# Notas / Contexto
Este sistema servirá de base para todas las futuras funcionalidades (dashboard, portafolio, etc.). Es el "Manual de Estilo" vivo del proyecto.
