# Descripción
El Hero debe ser el primer impacto que valide la autoridad de Conycom. Pasamos de un "Soluciones IT" genérico a un mensaje que evoque robustez (30 años de ERP farmacéutico) y modernidad (Laravel/IA).

# Objetivos
- [ ] **Titular de Impacto**: Reflejar la dualidad. Ej: "SOLIDEZ HEREDADA. AGILIDAD DIGITAL."
- [ ] **Subtitular con Datos**: Mencionar explícitamente los 30 años y la transición del software crítico.
- [ ] **Estética Técnica**: Mantener B/W pero introducir micro-elementos que sugieran "arquitectos de software" (líneas finas, grids, tipografía mono espaciada para detalles).
- [ ] **CTAs con Sentido**: Pasamos de "Agendar" a algo más directo sobre soluciones reales.

# Requisitos Estéticos
- Tipografía: Inter o roboto para el cuerpo, pero algo más "industrial" o "black" para el título.
- Colores: Negro absoluto, blancos crudos (#FBFBFB) y grises de interfaz técnica.
- Fondos: Subtítulos en cajas con bordes de 1px tipo "blueprint".

# Criterios de Aceptación
- [ ] El texto menciona la trayectoria de décadas.
- [ ] El diseño se siente premium y "caro", no una plantilla.
- [ ] El contraste asegura legibilidad extrema.

# Notas / Contexto
Inspirado en marcas de lujo técnico (tipo Rabbit, Framework, o agencias de diseño de alto nivel).
